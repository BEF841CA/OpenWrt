# OpenWrt-x86-64

自用 Openwrt 云自动编译x86-64固件

## 特点

- 自动拉取lede最新源码自动编译（每月一次）
- 纯净版无自带插件
- 默认使用argon主题

## 默认设置
地址：10.1.1.1
账号：root
密码：password
